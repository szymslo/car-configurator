# To run this app

### git clone https://github.com/szymslo/car-configurator
### cd car-configurator

## 1. Run Server
### cd backend
#### rm package-lock.json
#### npm install
#### npm start / yarn start

## 2. Run Client
### cd frontend
#### rm package-lock.json
#### npm install
#### npm start / yarn start
